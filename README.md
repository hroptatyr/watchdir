watchdir
========

+ github page: <https://github.com/hroptatyr/watchdir>

Usage:

    watchdir DIR...

File creation will come up as `c`, file deletion as `x`.
File moves as `ci` and `xo` pairs, if happening in the watched director{y,ies}.
